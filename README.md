# LAB-ASSIGNMENT-4 - Prompt Quizzer

A fully console-based JavaScript quiz game using `prompt()` and `alert()`.

## Features
- 7 questions with proper difficulty  
- Case-insensitive + trim() handling  
- Immediate feedback after every answer  
- Score + percentage calculation  
- Smart result message  
- Cancel quiz option included  
- Clean code using array of objects + function
